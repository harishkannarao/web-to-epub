# python exercise

Repository to explore python programming, testing and bundling of python programs

## Tools Required

* python `3.9`
* node `16.0.0`
* pipenv `2020.11.15`
* make `3.81`
* git `latest`
* pycharm `latest`

## Commands

### Install dependencies

    make init

### Run python script

    python main.py
    
### Verify flake8

    make flake8
    
### Create requirements.txt

    make requirements
